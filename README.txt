QuickMeal: Food Delivery App
Prototype of Food Ordering & Delivery Tracking Use Case

This prototype deals with the food ordering and tracking usecase described in Assignment 1.

1. Run the app: node app.js
2. Register new user if you are logging in the first time. After that you can sign in using the same credentials.
	
The prototype takes the customer through the restaurant selection,
menu selection, order checkout & payment and order status.
